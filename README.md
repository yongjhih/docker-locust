## Installation

```sh
curl -L https://github.com/yongjhih/docker-locust/raw/master/docker-locust > ~/bin/locust && \
chmod a+x ~/bin/locust
```

## Usage

Change to a directory which contains locustfile.py and optional requirements.txt

```
locust
```
